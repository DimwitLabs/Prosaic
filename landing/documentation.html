# Command line usage

$ prosaic [OPTIONS] [FILE]

# Options

--light / --dark Use light (default) or dark theme
--profile / --profiles Use or list profiles
--setup Run setup wizard again
--reference / --languages Show reference or list valid languages
--license Show MIT license
